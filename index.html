PROJECT 01

Web Application Penetration Testing

Black-box + Grey-box assessment of Dibimbing Shop e-commerce platform using OWASP WSTG methodology. Identified and documented critical vulnerabilities with full proof-of-concept exploitation.

OWASP WSTG Red Team Lab Env
WAPT-01 CRITICAL

SQL Injection — cart.php

SQL Injection discovered on the coupon code parameter in cart.php. Exploited via SQLmap to extract database contents. MySQL error exposed stack traces confirming injectable endpoint.

WAPT-01 :: Proof of Concept
REMEDIATION Prepared Statements + Strict Input Validation
WAPT-02 CRITICAL

Hardcoded Credentials (.env Exposure)

Admin credentials hardcoded in source code due to absence of environment variable configuration. Credentials discovered via exposed Git repository, granting full admin access.

WAPT-02 :: Proof of Concept
REMEDIATION Environment Variables + Credential Rotation
WAPT-03 HIGH

Stored XSS — Admin Panel Product URL

Stored Cross-Site Scripting vulnerability found in the Product URL field within the admin panel. Malicious payload persists and executes in victim browsers, enabling session hijacking.

WAPT-03 :: Proof of Concept
REMEDIATION Input Sanitization + Output Encoding + CSP Headers
WAPT-04 HIGH

Exposed Git Repository (.git Directory)

The .git directory is publicly accessible via the web root, exposing the full source code history, configuration files, and potentially sensitive commit data to unauthenticated users.

WAPT-04 :: Proof of Concept
REMEDIATION Block .git Access via Web Server Rules + Remove from Web Root
